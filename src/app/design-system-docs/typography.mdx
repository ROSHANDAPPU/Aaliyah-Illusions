# Typography

This document outlines the typography used in the application, defined in `tailwind.config.ts`.

## Font Sizes

The following font sizes are available, including responsive `clamp` values for headings and CTAs.

| Class Name            | Size (rem/px)                               | Description                                   |
| :-------------------- | :------------------------------------------ | :-------------------------------------------- |
| `text-xs`             | `0.75rem` (12px)                            | Extra small text                              |
| `text-sm`             | `0.875rem` (14px)                           | Small text                                    |
| `text-base`           | `1rem` (16px)                               | Base text size                                |
| `text-lg`             | `1.125rem` (18px)                           | Large text                                    |
| `text-xl`             | `1.25rem` (20px)                            | Extra large text                              |
| `text-2xl`            | `1.5rem` (24px)                             | 2x extra large text                           |
| `text-3xl`            | `1.875rem` (30px)                           | 3x extra large text                           |
| `text-4xl`            | `2.25rem` (36px)                            | 4x extra large text                           |
| `text-5xl`            | `3rem` (48px)                               | 5x extra large text                           |
| `text-6xl`            | `3.75rem` (60px)                            | 6x extra large text                           |
| `text-7xl`            | `4.5rem` (72px)                             | 7x extra large text                           |
| `text-8xl`            | `6rem` (96px)                               | 8x extra large text                           |
| `text-9xl`            | `8rem` (128px)                              | 9x extra large text                           |
| `text-clamp-h1-sm`    | `clamp(32px, 6vw, 88px)`                    | Responsive H1 for small screens               |
| `text-clamp-h1-lg`    | `clamp(48px, 10vw, 120px)`                  | Responsive H1 for large screens               |
| `text-clamp-h2`       | `clamp(28px, 4vw, 56px)`                    | Responsive H2                                 |
| `text-clamp-h3`       | `clamp(28px, 4vw, 52px)`                    | Responsive H3                                 |
| `text-clamp-cta`      | `clamp(14px, 1.2vw, 18px)`                  | Responsive Call to Action text                |
| `text-clamp-small`    | `clamp(10px, 1vw, 14px)`                    | Responsive small text                         |
| `text-clamp-wordmark-lg` | `clamp(48px, 12vw, 180px)`                 | Responsive wordmark for large displays        |
| `text-clamp-wordmark-hero` | `clamp(180px, 28vw, 480px)`                | Responsive wordmark for hero sections         |

The following font weights are available:

| `font-normal`   | `400`        |

| `font-semibold` | `600`        |
| `font-bold`     | `700`        |
| `font-extrabold`| `800`        |
| `font-black`    | `900`        |
