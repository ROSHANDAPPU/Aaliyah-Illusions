# Colors

This document outlines the color palette used in the application, defined in `tailwind.config.ts`.

## Primary Colors

| Variable        | Hex Value | Description                               |
| :-------------- | :-------- | :---------------------------------------- |
| `--primary`     | `#29CEF2` | Main accent color                         |
| `--primary-hover` | `#3DD8FC` | Hover state for primary elements          |

## Background Colors

| Variable            | Hex Value | Description                       |
| :------------------ | :-------- | :-------------------------------- |
| `--background-dark`   | `#050505` | Darkest background, used for main app background |
| `--background-medium` | `#0a0a0a` | Slightly lighter background       |
| `--background-light`  | `#111`    | Light background                  |
| `--background-lighter`| `#222`    | Even lighter background           |

## Neutral Text Colors

| Variable                | Hex Value | Description                       |
| :---------------------- | :-------- | :-------------------------------- |
| `--neutral-text-light`  | `#ffffff` | Light text, typically on dark backgrounds |
| `--neutral-text-medium` | `#888888` | Medium gray text                  |
| `--neutral-text-dark`   | `#4a4a4a` | Dark gray text                    |

## Gray Scale

| Variable    | Hex Value | Description                       |
| :---------- | :-------- | :-------------------------------- |
| `--gray-900`| `#18181b` | Very dark gray                    |
| `--gray-600`| `#71717a` | Darker gray                       |
| `--gray-500`| `#a1a1aa` | Medium gray                       |
| `--gray-400`| `#d4d4d8` | Lighter gray                      |
